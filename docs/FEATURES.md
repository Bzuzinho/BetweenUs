# Between Us — Features & Roadmap

### Ecrãs
- [x] Home com 3 portas de entrada (Casais / Solteiros / Relações Abertas)
- [x] Onboarding inteligente (3 passos: tipo → intenção → discrição)
- [x] Explorar perfis com Between Score e filtros
- [x] Soft Reveal (fotos desfocadas até match)

- [x] Sala Privada com chat funcional
- [x] Perfil com estatísticas (visitas, likes, matches)
- [x] Modo Invisível e Modo Discreto (toggles)
- [x] Bloco Premium
- [x] Between Guide (artigos educativos)
- [x] Animação de Match
- [x] Bilingue PT/EN

### Modais
- [x] Modo Acordo (regras de casal)
- [x] Mapa de Limites (Sim / Talvez / Não por categoria)
- [x] Consent Check (intenções antes de avançar)

## Between Score — Fatores de Compatibilidade

1. Tipo de relação compatível
2. Intenções alinhadas
3. Mapa de Limites sem conflitos
4. Disponibilidade e ritmo
5. Localização
6. Nível de discrição desejado
7. Verificação de perfil
8. Preferência por pessoa / casal / grupo

## Tipos de Perfil

| Tipo | Código |
|---|---|
| Solteiro/a | SINGLE |
| Casal | COUPLE |
| Casado/a | MARRIED |
| Relação aberta | OPEN |
| Poliamoroso/a | POLY |
| Casal à procura de terceira | COUPLE_THIRD |
| Casal à procura de casal | COUPLE_COUPLE |
| Curioso/a | CURIOUS |
